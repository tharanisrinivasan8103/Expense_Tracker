# Expense Tracker (React + Node/Express + MySQL)

## Setup

### Backend
1. cd backend
2. npm install
3. edit .env and set MySQL credentials
4. create database in MySQL: `CREATE DATABASE expense_tracker;`
5. npm run dev

### Frontend
1. cd frontend
2. npm install
3. npm start

The backend runs on port 5000 and the frontend on 3000.
